To run this file follow the steps bellow:
1) Clone the repository to VS code(you can use other editors; however, steps may vary, and compatibilaty is not garunteed)
2) ensure the file path is correct in your terminal and if it is not change it using CD. The correct file path should be to the folder in which app.py is directly inside
3) in a git terminal run: pip install flask
4) in the terminal run: py app.py
5) After completing step 4, something like this should appear in your terminal: Running on http://[some seris of numbers and periods]. Follow that link to run the project locally.
6) To quit type Ctrl and c in the terminal at the same time

That's it! Enjoy!
